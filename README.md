# Gloo AI Gateway Field Demo

![ai-gateway-main](images/ai-gateway-main.svg)

### Setup the Environment
```bash
./install-base.sh
```

### Navigate to the demo of your choice
```bash
cd demos
ls
```

### Run the Demo
```bash
./demo-script.sh
```
